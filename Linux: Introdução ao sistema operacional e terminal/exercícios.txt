Exercícios práticos de revisão

01. Abra o terminal.
02. Crie uma pasta de nome "Ubuntu" dentro da pasta "Documentos".
03. Mova esta pasta para o diretório pessoal.
04. Crie um arquivo vazio de nome "texte.txt" dentro da pasta "Ubuntu".
05. Renomeie este arquivo como "linux.txt".
06. Crie uma cópia deste arquivo na pasta "Documentos".
07. Exiba todos os comandos digitados no terminal.
08. Execute a ajuda do comando ls.
09. Execute o manual do comando mv.
10. Pare a execução do comando.
11. Saia do terminal utilizando sequência de teclas.
12. Exclua a pasta "Ubuntu".
13. Exclua o arquivo "linux.txt".
14. Limpe o terminal.
15. Utilize o comando para sair do terminal.
